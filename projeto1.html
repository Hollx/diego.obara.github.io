Resumo do Projeto

Este projeto utiliza dados financeiros de uma empresa fictícia para explorar padrões de despesas e receitas, identificar áreas de otimização, e prever tendências financeiras futuras.

Ferramentas Utilizadas

  • Python
  • Pandas para manipulação de dados
  • Matplotlib e Seaborn para visualizações
  • Machine Learning com scikit-learn

Resultados

A análise revelou padrões de despesas sazonais, além de prever um aumento de 10% nas vendas para o próximo trimestre. Um modelo de regressão linear foi usado para as previsões.
